# Case Reviews (Analyst Walkthroughs)

Case reviews are portfolio-ready analyst writeups generated from seeded runtime data and aligned to the Case File workflow.

## Sets
- Legacy long-form reviews: `docs/case_reviews/CASE_PLR_*.md`
- Demo review set (20 submitted): `docs/case_reviews/demo/CASE_PLR_*.md`
- Manifest/index: `docs/case_reviews/CASE_MANIFEST.json`

## What Each Review Includes
1. Case header (risk category, score, jurisdiction)
2. Signal-based findings and contradictory evidence
3. SQL evidence and trigger-check outputs
4. False-positive / false-negative checks
5. Decision rationale and analyst action
6. Tailored customer nudge copy
7. Follow-up plan and audit log summary
